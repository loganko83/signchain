## 🚨 현재 이슈

### 주요 문제
1. **서버 npm install 타임아웃**: 의존성 설치가 60초 내에 완료되지 않음
2. **signchain 서비스 중단**: PM2에서 signchain 프로세스 실행되지 않음

### 해결 방안
1. **npm install 최적화**:
   - `--production` 플래그로 dev dependencies 제외
   - `--cache-max=0` 옵션으로 캐시 문제 해결
   - 작은 단위로 나누어 설치

2. **대안 방법**:
   - 로컬에서 빌드 후 dist 폴더만 서버에 복사
   - Docker 컨테이너 사용 고려
